# ClashDisplay Font Files

This directory should contain the ClashDisplay font files in OTF format.

## Required Files:
- `ClashDisplay-Regular.otf`
- `ClashDisplay-Medium.otf`
- `ClashDisplay-Semibold.otf`
- `ClashDisplay-Bold.otf`

## Where to Get ClashDisplay:
ClashDisplay is available from:
- [Indian Type Foundry](https://www.indiantypefoundry.com/)
- [FontShare](https://www.fontshare.com/fonts/clash-display) (Free version)
- [GitHub](https://github.com/IndianTypeFoundry/Clash-Display)

## Installation:
1. Download the ClashDisplay font family
2. Place the OTF files in this directory (`assets/fonts/`)
3. Ensure the file names match exactly:
   - ClashDisplay-Regular.otf
   - ClashDisplay-Medium.otf
   - ClashDisplay-Semibold.otf
   - ClashDisplay-Bold.otf

## Font Usage in App:
- **ClashDisplay** (Primary): Used for all headings, titles, and display text
- **Montserrat** (Secondary): Used for body text, labels, and descriptions
